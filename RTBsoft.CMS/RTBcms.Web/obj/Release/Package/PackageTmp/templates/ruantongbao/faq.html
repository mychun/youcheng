产品介绍

Ping++ 是什么？

Ping++ 是一种全新的移动支付接入方式。通过使用 Ping++ 提供的 SDK，你的应用可以同时支持目前主流的多种移动支付渠道。同时 Ping++ 还提供交易管理平台，方便你统一管理所有支付渠道中的交易，有效降低你的前期开发和后期管理成本。

壹收款是什么？

「壹收款」是基于「Ping++ SDK」的全新支付产品，提供内嵌支付页面和完成页面，只需要一个按钮，一行代码，就可以实现一键收款。「壹收款」不仅进一步降低了开发的成本，同时进一步提高了支付流程体验，让你的用户更加专注在「支付」这件事上，轻松提高你的收入。

Ping++ 支持哪些渠道？

目前 Ping++ 支持：
　支付宝（移动支付、手机网页支付、PC 即时到帐和支付宝扫码支付）
　微信支付（针对 App 内的微信支付）、微信公众账号支付（微信扫码、服务号、红包、企业付款）
　银联移动支付（移动支付、手机网页支付和 PC 网关支付）
　百度钱包
　Apple Pay
　易宝支付
　京东支付
　应用内快捷支付境内卡（银联卡）
　应用内快捷支付境外卡（Visa/MasterCard/JCB）
你可任选其中一个或多个，如果您还有其他渠道的需求，请联系我们。

Ping++ 支持哪些语言？

目前支持 PHP, Ruby, Python, Java, Node.js, Go, C#.

Ping++ 支持哪些平台？

目前支持 Android, iOS, HTML 5.

产品使用

我可以使用吗？

目前 Ping++ 仅支持公司用户接入，任何需要在应用内加入支付模块的企业，都可以轻松接入 Ping++ 的 SDK。

怎么使用 Ping++ ？

  1. 注册 Ping++ 账号。
  2. 激活 Live Key。商户需要有效的营业执照和组织机构代码来激活 Ping++ Live Key 进行真实收款。
  3. 配置参数。在配置参数前，请确保你申请了以下渠道的支付产品并在有效期内：微信支付，微信公众账号支付，支付宝移动支付、支付宝手机网页支付、银联或百度钱包等渠道。查看如何自行申请渠道；
  4. 下载 SDK。点此下载；
  5. 技术联调。参数配置完成并下载相应的支付渠道 SDK 后，Ping++ 将为贵司配置一个专属的技术人员为你进行技术的对接；
  6. 上线收款。

我需要单独申请渠道吗？

目前 Ping++ 提供聚合 SDK，使用 Ping++ 前，你需要单独向各个支付渠道申请支付功能并获得渠道参数。查看如何自行申请渠道。你也可以通过 Ping++ 管理平台中的自动化流程提交代申请渠道的请求。

管理平台

管理平台有什么功能？

管理平台是 Ping++ 专为用户设计的应用管理及交易管理平台。你可以在 Ping++ 管理平台内管理应用、渠道配置、渠道申请、交易、退款以及下载报表。

如何查询交易？

可通过 Ping++ 管理平台查询所有支付渠道的交易详情。点击需要查询的应用，进入订单列表，输入订单编号即可查询此交易的渠道、交易成功时间和金额。

如何修改 App 名称？

登陆 Ping++ 管理平台，点击「应用信息」进行修改。在 Ping++ 平台上修改应用名称代表你在 Ping++ 平台上注册的应用名称已被修改，并不代表此应用名称已在支付渠道修改并更新。
说明：如果你需要修改你在渠道备注的 App 信息，从业务规则来讲，支付宝和银联支持修改 App 名称，微信不支持。但由于申请修改 App 名称需要一定的时间且手续繁琐， Ping++ 建议商户谨慎选择 App 名称，避免后续修改。

如何进行退款？

可以在 Ping++ 平台上直接进行退款。选择您需退款的订单，点击退款填入相应退款金额，可进行退款。
